random module imp functions
--------------------------
1.random()
  random.random()           =>gives a float number that ranges from 0.0 to 1.0
2.randint()
  random.randint(start,end) =>The random.randint() function generates a random integer from the range of numbers supplied.
3.randrange()
  random.randrange(start,end,step) =>The random.randrange() function selects an item randomly from the given range defined by the start, the stop, and the step parameters. By default, the start is set to 0. Likewise, the step is set to 1 by default.
4.choice()
  random.choice(string or list or tuple) =>The random.choice() function selects an item from a non-empty series at random.
5.shuffle()
  random.suffle(list or tuple)           =>The random.shuffle() function shuffles the given list randomly.

statistics module imp functions
-------------------------------
1.mean()
  statistics.mean(listvalues)           =>Gives mean of listed values
2.median()
  statistics.median(listvalues)         =>Gives median of listed values
3.mode()
  statistics.mode(listvalues)           =>Gives mode of listed values
4.stdev()
  statistics.stdev(listvalues)          =>Gives stdev of listed values
